Custom CSS and JavaScript with MyDiscord

Difficulty: 2/10
Recommended Tech-savviness: 2/10

In this guide, we learn how to install MyDiscord: A tool that injects custom CSS and JavaScript into the Discord client to allow custom themes and fun modifications to be applied.
