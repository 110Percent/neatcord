Custom CSS and JavaScript with MyDiscord

Learn how to customize your Discord client to add themes and scripts.

Let's be real. Discord doesn't exactly offer a lot of options when it comes to themeing. The only two choices you have are Light and Dark theme, and those only change the colours that show on your screen. Compact and Cozy mode exist for additional customization, but all it really does is adjust text and remove avatars.

Modifying CSS (and JS in certain aspects) allows you to customize your client however you want.
Justinoboyle's MyDiscord (based off leovoel's BeautifulDiscord) allows you to modify the CSS of your Discord client in real-time, as well as add JavaScript to the client.

If you already know what CSS is and how to use it, skip to the next section to get started. Otherwise, please continue normally. It's better to understand fully than be confused later.

What is CSS?

CSS, short for Cascading Style Sheet, is a very powerful styling language for the web. Essentially, it's what makes webpages pretty. Your browser reads CSS as a rule on how to display the webpage, from colours to fonts to animations. Without CSS, a lot of websites would be plain and boring.

Why am I applying this to Discord?

Discord's client runs on Electron, which uses Chromium browser as a base: The same base Google Chrome uses. It uses HTML, CSS and JS in a similar way Chrome and Firefox do. This means that applying these stylesheets to Discord will have the same effect as if you were applying them to a webpage.

Requirements

To complete this guide, you will need the following:

  • The most recent version of the Discord Desktop Client (Stable, PTB or Canary)
  • Python version 3 or higher
    • Note: When installing Python 3 with Windows, remember to check the "Add Python 3.x to PATH" checkbox on the installation dialog.
  • Pip Python Package Manager (preferrably the latest version)
Installing MyDiscord

If you have progressed this far, you should already have Python 3.5+ installed!

  1. Open the command line on your computer.
    • On Windows, this is named "Command Prompt" and can be opened from the Start/Windows menu.
    • On Mac and Linux, this should be named "Terminal" or "Terminal Emulator". You can find it using the application search (or spotlight in Mac).
    • Do NOT open the Python terminal. Open the regular command line for your operating system.
  2. Check to make sure you have the correct version of Python installed.
    • On Windows, enter python --version into the command prompt.
    • On Mac or Linux, type python3 --version into the terminal.
    • If the command returns an error or the Python version is a number under 3.5, install the latest version of Python.
  3. Open the Discord client.
    • In order to function, MyDiscord needs an instance of the Discord client to be running when it starts.
  4. Install MyDiscord using the command line.
    • On Windows, place the following into the command prompt:
      python -m pip install -U https://github.com/justinoboyle/MyDiscord/archive/master.zip
    • On Mac and Linux, place the following into the terminal:
      python3 -m pip install -U https://github.com/justinoboyle/MyDiscord/archive/master.zip

If all steps were completed as told, the installation should be complete.
Do NOT close the command line window!

Running MyDiscord

Running MyDiscord is easier than installing it. Since you already have your command line open, simply enter mydiscord. Your Discord client should restart itself, and you'll be good to go!

Editing CSS and JavaScript

When you ran MyDiscord, the script created two new files: discord-custom.css and discord-custom.js.

  • On Windows, these files can be located by opening the Run dialog (Windows + R) and entering %localappdata%\Discord\resources.
  • On Mac, these files can be found in /Applications/Discord.app/Contents/MacOS/resources.
  • On Linux, these files are located in the folder /usr/share/discord/resources.

Feel free to open discord-custom.css and edit it to your heart's delight. Any changes you save to this file should be automatically applied to the client.

In order to apply changes to the JavaScript, Discord must be reloaded by opening Discord and pressing Ctrl+R (Cmd+R on Mac).
